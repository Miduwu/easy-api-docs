Adds elements to the end of the array
## Parameters
|   Name   |            Description             | Enforced | Default Value |
|----------|------------------------------------|----------|---------------|
| Name     | The name of the array.             | Yes      | None          |
| Elements | Elements to be added to the array. | Yes      | None          |
## Example
```eats
$arrayPush[name;...elements]
```
## Function Aliases
- $arrayAppend
