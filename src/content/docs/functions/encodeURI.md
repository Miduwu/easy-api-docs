Encodes a string to URL format.
## Parameters
| Name |       Description       | Enforced | Default Value |
|------|-------------------------|----------|---------------|
| Text | The text to be encoded. | Yes      | None          |
## Example
```eats
$encodeURI[text]
```
## Function Aliases
- $encodeURIComponent
