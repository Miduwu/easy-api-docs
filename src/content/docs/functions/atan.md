Returns the arctangent of a number.
## Parameters
| Name  |                       Description                        | Enforced | Default Value |
|-------|----------------------------------------------------------|----------|---------------|
| Value | A numeric expression for which the arctangent is needed. | Yes      | None          |
## Example
```
$atan[value]
```
