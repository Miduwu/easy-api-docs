Returns whether the given text includes the provided word.
## Parameters
| Name |       Description       | Enforced | Default Value |
|------|-------------------------|----------|---------------|
| Text | The text to work with.  | Yes      | None          |
| Word | The word to be matched. | Yes      | None          |
## Example
```
$stringIncludes[text;word]
```
## Function Aliases
- $checkContains
- $includes
- $hasText
