Holds code execution for the given time.
## Parameters
|   Name   |             Description              | Enforced | Default Value |
|----------|--------------------------------------|----------|---------------|
| Duration | The duration to hold code execution. | Yes      | None          |
## Example
```eats
$wait[duration]
```
