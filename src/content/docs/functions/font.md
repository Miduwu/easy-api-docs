---
title: $font
description: Set the canvas font.
---

Set the canvas font.
:::danger
**$font** is only usable inside **$createCanvas**.
:::
## Parameters
|  Name  | Description  | Enforced | Default Value |
|--------|--------------|----------|---------------|
| Size   | Font size.   | Yes      | None          |
| Family | Font family. | No       | Arial         |
| Style  | Font style.  | No       |               |
## Example
```
$font[size;family?;style?]
```
