Creates a new object.
## Parameters
|   Name    |            Description             | Enforced | Default Value |
|-----------|------------------------------------|----------|---------------|
| Name      | The name for the new object.       | Yes      | None          |
| Structure | A key-value pair object interface. | No       | {}            |
## Example
```eats
$objectCreate[name;structure?]
```
## Function Aliases
- $createObject
- $newObject
