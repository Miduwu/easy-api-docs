Converts a text to lowercase.
## Parameters
| Name |        Description        | Enforced | Default Value |
|------|---------------------------|----------|---------------|
| Text | The text to be converted. | Yes      | None          |
## Example
```eats
$lowercase[text]
```
## Function Aliases
- $toLowerCase
- $lower
