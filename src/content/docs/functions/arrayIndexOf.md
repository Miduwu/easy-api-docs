Returns the index of the first array element matched.
## Parameters
| Name |      Description       | Enforced | Default Value |
|------|------------------------|----------|---------------|
| Name | The name of the array. | Yes      | None          |
| Word | Word to be found.      | Yes      | None          |
## Example
```eats
$arrayIndexOf[name;word]
```
