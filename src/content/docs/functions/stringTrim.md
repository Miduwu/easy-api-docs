---
title: $stringTrim
description: Trim spaces from start and end of the given text.
---

Trim spaces from start and end of the given text.
## Parameters
| Name |      Description       | Enforced | Default Value |
|------|------------------------|----------|---------------|
| Text | The text to work with. | Yes      | None          |
## Example
```
$stringTrim[text]
```
## Function Aliases
- $trim
- $trimSpace
