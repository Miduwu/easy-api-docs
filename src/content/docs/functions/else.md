else statement.
:::danger
**$else** is only usable inside **$stmt**.
:::
## Parameters
| Name |                     Description                     | Enforced | Default Value |
|------|-----------------------------------------------------|----------|---------------|
| Code | Code to execute if the past conditions are not met. | Yes      | None          |
## Example
```eats
$else[code]
```
