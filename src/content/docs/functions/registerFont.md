Register a font in your system.
## Parameters
|  Name  | Description  | Enforced | Default Value |
|--------|--------------|----------|---------------|
| Family | Font family. | Yes      | None          |
| Path   | Font path.   | Yes      | None          |
## Example
```
$registerFont[family;path]
```
