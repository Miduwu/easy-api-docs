Joins an array using the provided separator.
## Parameters
|   Name    |           Description           | Enforced | Default Value |
|-----------|---------------------------------|----------|---------------|
| Name      | The name of the array.          | Yes      | None          |
| Separator | Character to join the elements. | No       | ,             |
## Example
```eats
$arrayJoin[name;separator?]
```
