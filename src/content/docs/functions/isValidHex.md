Returns wheter the text inside is a valid hex code.
## Parameters
| Name |      Description       | Enforced | Default Value |
|------|------------------------|----------|---------------|
| Text | The text to be tested. | Yes      | None          |
## Example
```
$isValidHex[text]
```
