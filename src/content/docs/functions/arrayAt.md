---

Returns the array element at the provided index.
## Parameters
| Name  |            Description            | Enforced | Default Value |
|-------|-----------------------------------|----------|---------------|
| Name  | The name of the array.            | Yes      | None          |
| Index | The element index to be accessed. | Yes      | None          |
## Example
```eats
$arrayAt[name;index]
```
