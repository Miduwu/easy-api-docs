Executes a code field for each array element.
## Parameters
| Name |      Description       | Enforced | Default Value |
|------|------------------------|----------|---------------|
| Name | The name of the array. | Yes      | None          |
| Code | Code to be executed.   | Yes      | None          |
## Example
```eats
$arrayForEach[name;code]
```
