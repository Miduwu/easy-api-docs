Evaluates easy-api.ts code.
## Parameters
|    Name     |                    Description                    | Enforced | Default Value |
|-------------|---------------------------------------------------|----------|---------------|
| Return Code | Whether return evaluation result. (boolean value) | Yes      | None          |
| Code        | Native code to be evaluated.                      | Yes      | None          |
## Example
```eats
$eval[returnCode;code]
```
