Creates a native custom function.
## Parameters
| Name |      Description      | Enforced | Default Value |
|------|-----------------------|----------|---------------|
| Name | Native function name. | Yes      | None          |
| Code | Native function code. | Yes      | None          |
## Example
```eats
$createCallback[name;...code]
```
## Function Aliases
- $function
- $newFunction
- $newCallback
- $createFunction
