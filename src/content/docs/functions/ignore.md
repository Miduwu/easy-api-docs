Ignores the code inside.
## Parameters
|  Name  |            Description             | Enforced | Default Value |
|--------|------------------------------------|----------|---------------|
| Code   | Code to be ignored.                | Yes      | None          |
| Return | Whether return the original input. | No       | false         |
## Example
```
$ignore[code;return?]
```
