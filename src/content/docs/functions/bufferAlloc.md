Allocates a buffer.
## Parameters
| Name  |      Description       | Enforced | Default Value |
|-------|------------------------|----------|---------------|
| Name  | The name of the array. | Yes      | None          |
| Bytes | Buffer byte length.    | Yes      | None          |
## Example
```
$bufferAlloc[name;bytes]
```
