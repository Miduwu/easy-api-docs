Evaluates JavaScript code.
## Parameters
|    Name     |                    Description                    | Enforced | Default Value |
|-------------|---------------------------------------------------|----------|---------------|
| Return Code | Whether return evaluation result. (boolean value) | Yes      | None          |
| Code        | JavaScript code to be evaluated.                  | Yes      | None          |
## Example
```eats
$jsEval[returnCode;...code]
```
## Function Aliases
- $js
