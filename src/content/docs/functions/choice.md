title: $choice
description: Returns a random text from all possible options.
---

Returns a random text from all possible options.
## Parameters
| Name  |              Description              | Enforced | Default Value |
|-------|---------------------------------------|----------|---------------|
| Texts | The texts to set as possible options. | Yes      | None          |
## Example
```
$choice[...texts]
```
## Function Aliases
- $randomText
- $randomChoice
