Returns e (the base of natural logarithms) raised to a power.
## Parameters
| Name  |                    Description                    | Enforced | Default Value |
|-------|---------------------------------------------------|----------|---------------|
| Value | A numeric expression representing the power of e. | Yes      | None          |
## Example
```eats
$exp[value]
```
