Returns the arc cosine (or inverse cosine) of a number.
## Parameters
| Name  |      Description      | Enforced | Default Value |
|-------|-----------------------|----------|---------------|
| Value | A numeric expression. | Yes      | None          |
## Example
```eats
$acos[value]
```
