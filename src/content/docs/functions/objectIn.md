Check if an object exists in the environment data.
## Parameters
| Name |         Description          | Enforced | Default Value |
|------|------------------------------|----------|---------------|
| Name | The name of the object.      | Yes      | None          |
| Path | Property path to be checked. | Yes      | None          |
## Example
```
$objectIn[name;path]
```
## Function Aliases
- $objectHas
