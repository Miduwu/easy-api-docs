Executes code synchronously, so, the interpreter won't wait for its execution.
## Parameters
| Name |     Description      | Enforced | Default Value |
|------|----------------------|----------|---------------|
| Code | Code to be executed. | Yes      | None          |
## Example
```eats
$sync[code]
```
