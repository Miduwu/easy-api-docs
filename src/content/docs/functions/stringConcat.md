Converts a text to lowercase.
## Parameters
| Name  |          Description          | Enforced | Default Value |
|-------|-------------------------------|----------|---------------|
| Text  | The text to be converted.     | Yes      | None          |
| Texts | The texts to be concatenated. | Yes      | None          |
## Example
```eats
$stringConcat[text;...texts]
```
## Function Aliases
- $concat
- $concatString
