Logs something in the console.
## Parameters
| Name  |            Description             | Enforced | Default Value |
|-------|------------------------------------|----------|---------------|
| Texts | Texts to be logged in the console. | Yes      | None          |
## Example
```eats
$log[...texts]
```
## Function Aliases
- $print
- $echo
