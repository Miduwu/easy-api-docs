---
title: $deleteFile
description: Unlinks a path/file from the file system.
---

Unlinks a path/file from the file system.
## Parameters
| Name |             Description             | Enforced | Default Value |
|------|-------------------------------------|----------|---------------|
| Path | The path of the file to be deleted. | Yes      | None          |
## Example
```
$deleteFile[path]
```
## Function Aliases
- $unlinkFile
