---
title: $bufferCreate
description: Creates a new buffer.
---

Creates a new buffer.
## Parameters
|  Name   |      Description       | Enforced | Default Value |
|---------|------------------------|----------|---------------|
| Name    | The name of the array. | Yes      | None          |
| Content | Buffer content.        | Yes      | None          |
## Example
```
$bufferCreate[name;content]
```
