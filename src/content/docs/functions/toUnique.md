Creates an string of unique values.
## Parameters
|   Name    |          Description           | Enforced | Default Value |
|-----------|--------------------------------|----------|---------------|
| Separator | Elements and result separator. | Yes      | None          |
| Elements  | Elements to filter.            | Yes      | None          |
## Example
```eats
$toUnique[separator;elements]
```
