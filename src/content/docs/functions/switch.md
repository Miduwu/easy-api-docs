Executes a switch statement.
## Parameters
| Name  |      Description       | Enforced | Default Value |
|-------|------------------------|----------|---------------|
| Value | Value to be evaluated. | Yes      | None          |
| Cases | Cases to be executed.  | Yes      | None          |
## Example
```
$switch[value;cases]
```
