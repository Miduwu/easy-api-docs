---
title: $opacity
description: Set the canvas opacity.
---

Set the canvas opacity.
:::danger
**$opacity** is only usable inside **$createCanvas**.
:::
## Parameters
|  Name  |          Description          | Enforced | Default Value |
|--------|-------------------------------|----------|---------------|
| Amount | Context amount to be applied. | Yes      | None          |
## Example
```
$opacity[amount]
```
