Returns the absolute value of a number.
## Parameters
| Name  |             Description             | Enforced | Default Value |
|-------|-------------------------------------|----------|---------------|
| Value | Value to extract the absolute from. | Yes      | None          |
## Example
```eats
$abs[value]
```
