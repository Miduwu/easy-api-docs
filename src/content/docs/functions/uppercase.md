Converts a text to uppercase.
## Parameters
| Name |        Description        | Enforced | Default Value |
|------|---------------------------|----------|---------------|
| Text | The text to be converted. | Yes      | None          |
## Example
```eats
$uppercase[text]
```
## Function Aliases
- $toUpperCase
- $upper
