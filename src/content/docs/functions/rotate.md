Rotates a canvas.
:::danger
**$rotate** is only usable inside **$createCanvas**.
:::
## Parameters
| Name  |   Description   | Enforced | Default Value |
|-------|-----------------|----------|---------------|
| Angle | Rotation angle. | Yes      | None          |
## Example
```eats
$rotate[angle]
```
