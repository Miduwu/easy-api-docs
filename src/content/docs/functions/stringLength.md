Returns the character count of the provided text.
## Parameters
| Name |      Description       | Enforced | Default Value |
|------|------------------------|----------|---------------|
| Text | The text to work with. | Yes      | None          |
## Example
```eats
$stringLength[text]
```
## Function Aliases
- $charCount
