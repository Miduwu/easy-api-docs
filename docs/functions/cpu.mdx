# $cpu
Returns how many CPU resource is used.

## Usage
> `$cpu: [decimals?]`

## Fields
| Name         | Description                                | Type   | Level    |
| -----------  | ------------------------------------------ | ------ | -------- |
| decimals     | How many decimals for the return number.   | String | Optional |

## Example
```yaml
$cpu:[2] # 0.27
```
