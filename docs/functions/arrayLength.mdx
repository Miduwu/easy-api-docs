# $arrayLength
Returns the array length.

## Usage
> `$arrayLength:[name]`

## Fields
| Name  | Description          | Type   | Level    |
| ----- | -------------------- | ------ | -------- |
| name  | The array name.      | String | Required |

## Example
```yaml
$newArray:[data;a,b,c,d;,]
$arrayLength:[data] # 4
```
