# $arrayDelete
Deletes an array from the interpreter data.

## Usage
> `$arrayDelete:[name]`

## Fields
| Name  | Description        | Type   | Level    |
| ----- | ------------------ | ------ | -------- |
| name  | The array name.    | String | Required |

## Example
```yaml
$newArray:[data;a,b,c,d;,]
$arrayAt:[data;3] # c
$arrayDelete:[data]
$arrayAt:[data;3] # throws an error.
```
