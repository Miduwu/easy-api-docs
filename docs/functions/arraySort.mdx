# $arraySort
Sorts an array alphabetically.

## Usage
> `$arraySort:[name]`

## Fields
| Name  | Description          | Type   | Level    |
| ----- | -------------------- | ------ | -------- |
| name  | The array name.      | String | Required |

## Example
```yaml
$newArray:[data;d,b,a,c;,]
$arraySort:[data]
$arrayJoin:[data;, ] # a, b, c, d
```
