# $isObject
Check if the inside is a object or not.

### Parameters
| Name           | Type                | Description                                  | Optional |
| -------------- | ------------------- | -------------------------------------------- | -------- |
| variable name  | String              | The variable name where the object is saved. | false    |

### Example
```js
$var[mine;{}]
$isObject[mine] // Returns: true
```
