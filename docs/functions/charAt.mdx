# $charAt
Returns the character at the provided index.

## Usage
> `$charAt: [text;index]`

## Fields
| Name  | Description           | Type   | Level    |
| ----- | --------------------- | ------ | -------- |
| text  | The text to analize.  | String | Required |
| index | The char index.       | String | Required |

## Example
```yaml
$charAt:[HELLO WORLD;7] # W
```
