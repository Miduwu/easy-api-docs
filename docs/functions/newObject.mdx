# $newObject
Creates a new object.

## Usage
> `$newObject:[name;object]`

## Fields
| Name          | Description        | Type   | Level    |
| ------------- | ------------------ | ------ | -------- |
| name          | The object name.   | String | Required |
| object        | The object itself. | String | Required |

## Example
```yaml
$newObject:[data;{ "message": "hi", "props": [{ "name": "CGX" }, { "name": "Mid" }] }]
```
