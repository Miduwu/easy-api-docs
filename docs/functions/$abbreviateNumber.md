# $abbreviateNumber
Abbreviates the provided number.

### Parameters:
| Name        | Type        | Description                            | Optional |
| ----------- | ----------- | ---------------------------------------| -------- |
| number      | Number      | Number to abbreviate.                  | false    |
| decimals    | Number      | How many decimals for the abbreviation.| true     |

### Example:
```js
$abbreviateNumber[9210;3] //Returns: 9.21k
```
