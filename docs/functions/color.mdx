# $color
Sets the canvas context color.

:::caution

This function __just can be used__ inside **[$createCanvas](createCanvas)**

:::

## Usage
> `$color:[color code]`

## Fields
| Name          | Description                       | Type	     | Level    |
| ------------- | --------------------------------- | ---------- | -------- |
| color         | The hex color code.               | String     | Required |

## Example
```yaml
$createCanvas: [
	# ...
	$color: [040E20]
	# ...
]
```
