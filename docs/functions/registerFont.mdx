# $registerFont
Register a new canvas font (.ttf/.otf).

## Usage
> `$registerFont:[family;path]`

## Fields
| Name        | Description                           | Type	   | Level    |
| ----------- | ------------------------------------- | ---------- | -------- |
| family      | The font family/name.                 | String	   | Required |
| path        | The font path.						  | String     | Required |

## Example
```yaml
$registerFont: [myFont;./assets/fonts/myFont.ttf]
```
