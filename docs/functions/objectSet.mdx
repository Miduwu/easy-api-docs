# $objectSet
Set a property in the given object.

## Usage
> `$objectSet: [name;path;value]`

## Fields
| Name  | Description           | Type   | Level    |
| ----- | --------------------- | ------ | -------- |
| name  | The object name.      | String | Required |
| sep   | Values separator.     | String | Optional |

## Example
```yaml
$newObject:[data;{
    "moonlightgroup": {
        "packages": [{
            "name": "easy-api.ts",
            "description": "A string package to create APIs with ease."
        },{
            "name": "erine",
            "description": "A powerul command framework to create discord bots with ease."
        }],
        "members": []
    }
}]
$objectSet:[data;moonlightgroup.members[0];Mid]
$objectSet:[data;moonlightgroup.members[1];CGX]
$getObject:[data] # {"moonlightgroup":{"packages":[{"name":"easy-api.ts","description":"A string package to create APIs with ease."},{"name":"erine","description":"A powerul command framework to create discord bots with ease."}],"members": ["Mid", "CGX"]}}
```
