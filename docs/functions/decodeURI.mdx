# $decodeURI
Decodes a URI text.

## Usage
> `$decodeURI:[text]`

## Fields
| Name  | Description           | Type   | Level    |
| ----- | --------------------- | ------ | -------- |
| text  | The text to decode.   | String | Required |

## Example
```yaml
$decodeURI:[Hi%20world,%20from%20eats.] # Hi world, from eats.
```
