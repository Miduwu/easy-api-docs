# $deleteFile
Deletes a file, if the file exists.

### Parameters:
| Name        | Type        | Description                                       | Optional |
| ----------- | ----------- | --------------------------------------------------| -------- |
| path        | String      | The location from the file will be deleted.       | false    |

### Example:
```js
$deleteFile[./assets/files/best.txt] //Deletes "best.txt" file from "./assets/files".
```
