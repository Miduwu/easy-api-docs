# $root
Join your files name to the process cwd.

## Usage
> `$root:[...paths]`

## Fields
| Name       | Description             | Type   | Level    |
| ---------- | ----------------------- | ------ | -------- |
| paths      | Paths to join with cwd. | String | Required |

## Example
```yaml
$root:[./file.txt] // C:\Project\eats\hi\file.txt
```
