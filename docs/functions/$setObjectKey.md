# $setObjectKey
Set a key value to the created object.

### Parameters:
| Name      | Type                | Description                        |
| --------- | ------------------- | ---------------------------------- |
| name | String              | The property name. |
| value | Any              | The value of this property. |

### Example:
```js
$setObjectKey[text;hello] 
$createObject // required to use this function
```
