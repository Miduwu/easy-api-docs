# $and
Checks whether all conditions are met.

## Usage
> `$and:[...conditions]`

## Fields
| Name        | Description          | Type   | Level    |
| ----------- | -------------------- | ------ | -------- |
| conditions  | Conditions to check. | String | Required |

## Example
```js
$and:[1==1;1<2;3>2] // true
```
