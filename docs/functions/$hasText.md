# $hasText
Check if a code has a text.

### Parameters:
| Name        | Type        | Description                          | Optional |
| ----------- | ----------- | ------------------------------------ | -------- |
| source      | String      | The source text.                     | false    |
| ...texts    | String      | The texts to check.                  | false    |

### Example:
```js
$hasText[hello world;uno;fella;ew] // false
```
