# $encodeURI
Encodes a URI text.

## Usage
> `$encodeURI:[text]`

## Fields
| Name  | Description           | Type   | Level    |
| ----- | --------------------- | ------ | -------- |
| text  | The text to encode.   | String | Required |

## Example
```js
$encodeURI:[Hi world, from eats.] // Hi%20world,%20from%20eats.
```
