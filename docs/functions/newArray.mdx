# $newArray
Creates a new array by spliting a text.

## Usage
> `$newArray:[name;elements;separator?]`

## Fields
| Name | Description | Type | Level |
| ---- | ----------- | ---- | ---- |
| name | The array name. | String | Required |
| elements | Array elements. | String | Required |
| separator | Element separator. | String | Optional |

## Example
```js
$newArray:[fruits;Banana-Apple-Watermelon-Mango;-]
```
