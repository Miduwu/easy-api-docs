# $arraySlice

### Parameters
| Name           | Type                | Description                                  | Optional |
| -------------- | ------------------- | -------------------------------------------- | -------- |
| variable name  | String              | The variable name where the array is saved.  | false    |
| start          | String              | Element of the split.                        | false    |
| join           | String              | Something to replace the element.            | true     |
