# $arrayAt
Return the array element at the provided index.

## Usage
> `$arrayAt:[name;index]`

## Fields
| Name | Description | Type | Level |
| ---- | ----------- | ---- | ---- |
| name | The array name. | String | Required |
| index | The element index. | Number | Required |

## Example
```js
$newArray:[data;a,b,c,d;,]
$arrayAt:[data;3] // c
```
