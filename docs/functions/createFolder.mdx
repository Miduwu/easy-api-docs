# $createFolder
Creates a folder at the provided path.

## Usage
> `$createFolder:[path]`

## Fields
| Name        | Description                               | Type   | Level    |
| ----------- | ----------------------------------------- | ------ | -------- |
| path        | Folder path.                              | String | Required |

## Example
```yaml
$createFolder: [./assets]
// The folder "assets" will be created.
```
