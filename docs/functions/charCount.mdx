# $charCount
Returns the character count of the provided text.

## Usage
> `$charCount: [text]`

## Fields
| Name  | Description           | Type   | Level    |
| ----- | --------------------- | ------ | -------- |
| text  | The text to analize.  | String | Required |

## Example
```yaml
$charCount:[HELLO WORLD] # 11
```
