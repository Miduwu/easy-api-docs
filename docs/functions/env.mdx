# $env
Get an enviorement variable value (You need to authorize that).

## Usage
> `$env: [name;authorization]`

## Fields
| Name          | Description                    | Type    | Level    |
| ------------- | ------------------------------ | ------- | -------- |
| name          | The environment variable name. | String  | Required |
| authorization | Your authorization (true).     | Boolean | Required |

## Example
```yaml
$env: [superSecretKey;true]
```
