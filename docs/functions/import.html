$import

Import something from a file.

Usage​

$import: [path;property?]

Fields​

NameDescriptionTypeLevel
pathThe file path.StringRequired
propertyThe property to import.StringRequired

Example​

# ./file.module.js
# FILE CONTENT:
# const object = {
# name: "EASY-API.TS",
# description: "A string package to create APIs with ease."
# }
$import: [./file.module.js;object.name] # EASY-API.TS
