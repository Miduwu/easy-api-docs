# $arrayFindIndex
Finds the index of the provided array element.

## Usage
> `$arrayExists:[name;match]`

## Fields
| Name  | Description          | Type   | Level    |
| ----- | -------------------- | ------ | -------- |
| name  | The array name.      | String | Required |
| match | The element to find. | String | Required |

## Example
```yaml
$newArray:[data;a,b,c,d;,]
$arrayFindIndex:[data;d] # 4
$arrayFindIndex:[data;mango] # -1 (Doesn't exists)
```
