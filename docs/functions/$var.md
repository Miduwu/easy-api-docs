# $var
Set a new local variable value.

### Parameters:
| Name        | Type        | Description                          | Optional |
| ----------- | ----------- | ------------------------------------ | -------- |
| name        | String      | The variable name to get the value.  | false    |
| value       | String      | The value this variable will have.   | false    |

### Example:
```js
// now you can get the value using $get
$var[test;ok]
```
