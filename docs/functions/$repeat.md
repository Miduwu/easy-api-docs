# $repeat
Repeat a text X times.

### Parameters:
| Name        | Type        | Description                          | Optional |
| ----------- | ----------- | ------------------------------------ | -------- |
| text        | String      | The text to repeat.                  | false    |
| times       | Number      | The times to repeat that text.       | false    |

### Example:
```js
$repeat[yo;3] // yoyoyo
```
