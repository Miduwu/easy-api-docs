# $request

 Make an http request with GET method.

### Parameters:
| Name | Type | Description |
| ---- | ---- | ------------|
|  url | String |The link to make the request. |

### Example:

```js
$request[https://api.miduwu.ga/json/translate?source=auto&target=fr&text=Hello+world]
```
