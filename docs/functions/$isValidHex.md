# $isValidHex
Check if a text is a valid hex color code.

### Parameters:
| Name      | Type        | Description              | Optional    |
| --------- | ----------- | ------------------------ | ----------- |
| text      | String      | The possible color code. | false       |

### Example:
```js
$isValidHex[#040E20] // true
$isValidHex[white] // false
 ```
