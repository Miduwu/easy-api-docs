# $appendContent
Appends content to the end of the provided file.

## Usage
> `$appendContent: [path;content]`

## Fields
| Name        | Description                               | Type   | Level    |
| ----------- | ----------------------------------------- | ------ | -------- |
| path        | File path.                                | String | Required |
| content     | Content to append to the file.            | String | Required |

## Example
```js
$set:[content;Hiii, good morning!]
$appendContent:[./file.txt;$get:[content]]
```
