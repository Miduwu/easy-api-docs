# $fileExists
Check if the provided file exists at the given path.

## Usage
> `$fileExists: [path]`

## Fields
| Name        | Description                               | Type   | Level    |
| ----------- | ----------------------------------------- | ------ | -------- |
| path        | File path.                                | String | Required |

## Example
```yaml
$fileExists: [./file.txt] // true
```
