# $redirect
Redirects the request to another API path.

### Parameters:
| Name           | Type                | Description                                  | Optional |
| -------------- | ------------------- | -------------------------------------------- | -------- |
| API path / URL | String              | The link to redirect the request.            | false    |

### Example:
```js
$redirect[https:\/\/npmjs.com/package/easy-api.ts]
```
