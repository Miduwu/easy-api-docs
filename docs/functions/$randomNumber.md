# $randomNumber
Get a random number between two numbers.

### Parameters:
| Name        | Type        | Description                          | Optional |
| ----------- | ----------- | ------------------------------------ | -------- |
| min         | Number      | The min number.                      | false    |
| max         | Number      | The max number.                      | false    |

### Example:
```js
$randomNumber[1;5] // 3
```
