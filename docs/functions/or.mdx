# $or
Checks whether at least one conditions is met.

## Usage
> `$or:[...conditions]`

## Fields
| Name        | Description          | Type   | Level    |
| ----------- | -------------------- | ------ | -------- |
| conditions  | Conditions to check. | String | Required |

## Example
```yaml
$or: [1==0;2<1;0==0] # true
$or: [1==0;2<1;0==1] # false
$or: [undefined;null;1] # 1
$or: [undefined;false;3>2;null] # true
```
