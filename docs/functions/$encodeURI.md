# $encodeURI
Encode a text to URI components.

### Parameters:
| Name        | Type        | Description                          | Optional |
| ----------- | ----------- | ------------------------------------ | -------- |
| text        | String      | The text to encode.                  | false    |

### Example:
```js
$encodeURI[Hello lol, hru?] // Hello%20lol%2C%20hru%3F
```
