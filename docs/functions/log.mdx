# $log
Logs something in the console.

## Usage
> `$log:[...messages]`

## Fields
| Name       | Description          | Type   | Level    |
| ---------- | -------------------- | ------ | -------- |
| messages   | The messages to log. | String | Required |

## Example
```js
$log:[Hello;world;from;eats!] // Hello world from eats!
```
