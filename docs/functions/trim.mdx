# $trim
Trim the start and end of the text.

## Usage
> `$trim:[text]`

## Fields
| Name    | Description           | Type   | Level    |
| ------- | --------------------- | ------ | -------- |
| text    | The text to trim.     | String | Required |

## Example
```js
$trim: [                     Hi, how are you?               ] // Hi, how are you?
```
