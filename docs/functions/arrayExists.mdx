# $arrayExists
Checks if an array exists or not.

## Usage
> `$arrayExists:[name]`

## Fields
| Name  | Description        | Type   | Level    |
| ----- | ------------------ | ------ | -------- |
| name  | The array name.    | String | Required |

## Example
```yaml
$arrayExists:[data] # false
$newArray:[data;a,b,c,d;,]
$arrayExists:[data] # true
```
