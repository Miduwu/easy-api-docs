# $set
Set a environment variable.

## Usage
> `$set:[name;value]`

## Fields
| Name  | Description           | Type   | Level    |
| ----- | --------------------- | ------ | -------- |
| name  | Variable name.        | String | Required |
| value | Variable value.       | String | Required |

## Example
```js
$set:[greet;Hi, good morning.]
```
