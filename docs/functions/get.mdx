# $get
Get a environment variable.

## Usage
> `$get:[name]`

## Fields
| Name  | Description           | Type   | Level    |
| ----- | --------------------- | ------ | -------- |
| name  | Variable name.        | String | Required |

## Example
```js
$set:[greet;Hi, good morning.]
$get:[greet] // Hi, good morning.
```
