# $readFile
Reads a file if the file exists.

## Usage
> `$readFile: [path]`

## Fields
| Name        | Description                               | Type   | Level    |
| ----------- | ----------------------------------------- | ------ | -------- |
| path        | File path.                                | String | Required |

## Example
```yaml
$readFile: [./file.txt] # Hello, good morning!
```
