## Using Escapers
### How does "escapers" work?
Those 'variables/escapers' will be replaced in our code to the specified value, like so: `@semi` => `;`
In aoi.js they are defined like: `#ANYTHING#`
### Example
```
@semi) => ;)
```
### List
| Symbol       | Replacement |
| ------------ | ----------- |
| **@left**    | ]           |
| **@right**   | [           |
| **@semi**    | ;           |
| **@colon**   | :           |
| **@space**   |             |
| **@at**      | @           |
